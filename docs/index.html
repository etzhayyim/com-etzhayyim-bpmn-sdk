🎯 Overview

@etzhayyim/bpmn-sdk is a comprehensive TypeScript SDK for enterprise BPMN workflow development. It provides complete BPMN 2.0 support with a type-safe DSL, seamless bpmn-engine integration, and enterprise-grade features.

✨ Key Features

🎨 Complete BPMN 2.0 Support

Type-safe representation of all BPMN 2.0 elements including events, gateways, tasks, and subprocesses.

⚡ TypeScript DSL

Declarative process modeling with full IDE support and compile-time validation.

🚀 Runtime Integration

Seamless integration with bpmn-engine for process execution and monitoring.

🔄 Bidirectional Conversion

XML ↔ TypeScript round-trip conversion for model interchange.

✅ Static Validation

Reachability analysis and structural verification at build time.

👥 Human Task Management

SLA management, escalation, and workflow integration for human tasks.

🧪 Property Testing

Automated test generation and formal validation of process properties.

📊 Operational Monitoring

OpenTelemetry integration and observability for enterprise deployments.

🚀 Quick Start

# Install the SDK
pnpm add @etzhayyim/bpmn-sdk

# Define a process with TypeScript DSL
import { flow } from '@etzhayyim/bpmn-sdk/dsl';

const invoiceProcess = flow('InvoiceApproval', f => f
  .process('InvoiceApproval', p => p
    .startEvent('StartEvent')
    .userTask('ReviewInvoice')
    .exclusiveGateway('AmountCheck')
    .serviceTask('AutoApprove')
    .serviceTask('ProcessInvoice')
    .endEvent('EndEvent')
    .sequenceFlow('StartEvent', 'ReviewInvoice')
    .sequenceFlow('ReviewInvoice', 'AmountCheck')
    .sequenceFlow('AmountCheck', 'AutoApprove')
      .condition('${amount <= 1000}')
    .sequenceFlow('AmountCheck', 'ProcessInvoice')
      .condition('${amount > 1000}')
    .sequenceFlow('AutoApprove', 'ProcessInvoice')
    .sequenceFlow('ProcessInvoice', 'EndEvent')
  )
);

// Deploy and execute
const { runtime, context } = await deployAndStart(invoiceProcess, {
  variables: { amount: 500, invoiceId: 'INV-001' },
  businessKey: 'PROC-001'
});

📋 BPMN 2.0 Coverage

Complete support for BPMN 2.0 specification including:

  • Events: Start, Intermediate, End, Boundary events
  • Tasks: Service, User, Manual, Script, Business Rule, Send, Receive, Call Activity
  • Gateways: Exclusive, Inclusive, Parallel, Event-based, Complex
  • Subprocesses: Embedded, Reusable, Event, Transaction, Ad-hoc
  • Data & Artifacts: Data Objects, Stores, Message Flows, Associations

🏗️ Architecture

The SDK is organized into focused packages:

  • core: BPMN 2.0 type definitions and internal representation
  • dsl: Declarative process modeling with TypeScript
  • compiler: IR to BPMN XML conversion
  • runtime: bpmn-engine integration and execution
  • human: Human task management and SLA
  • validation: Static validation and analysis
  • testing: Property-based testing framework
  • ops: OpenTelemetry monitoring and operations
